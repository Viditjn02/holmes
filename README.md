# HOLMES

Point at a company. A swarm of AI detectives finds the live communities where its buyers are asking the exact question you answer — each as a clickable, intent-scored thread + a drafted reply you approve — and generates a video ad. Live, in under 3 minutes.

The atomic deliverable is **a clickable, intent-scored link to a live conversation** (not a score, not a contact). Built on Convex (realtime swarm) + Exa (the thread moat) + Veo (video) + OpenAI.

> Setup, architecture, and run instructions: see the full README (built by the docs agent) and `.env.example`.
